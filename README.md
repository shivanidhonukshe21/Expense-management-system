# Expense Management System

A modern and responsive Expense Management web application built using **React.js** and **Vite**. 
This project helps users manage daily expenses, track spending, and 
organize financial records in a clean and user-friendly interface.

## 🚀 Features
- Add new expenses easily
- View expense history
- Track amount, category, and date
- Delete unwanted records
- Responsive modern UI
- Fast performance using Vite
- Clean component-based structure

## 🛠️ Tech Stack
- React.js
- JavaScript
- Vite
- CSS
- HTML5


⚙️ Installation & Setup
1.Clone repository
git clone <repository-link>

2.Open project folder
cd expense-management

3.Install dependencies
npm install

4.Run project
npm run dev

5.Open browser
http://localhost:5173
